1 окно даем название узлу и используем конфигурацию для создания генезис блока:   

geth --identity  "yourldentity" --datadir "./" init ./genesis.json

1 окно запускаем узел: 

Для новой версии Git Bash(использовать эту):

geth --networkid 12321 --http --http.port="8545" --http.corsdomain="http://remix.ethereum.org" --http.api="web3,eth,debug,personal,net" --vmdebug --datadir "./" --dev --allow-insecure-unlock

Для старой версии Git Bash:

geth --datadir "./" --networkid 30129 --rpc  --rpcport "8545"  --rpcapi="web3,eth,debug,personal,net"  --rpccorsdomain "*" --allow-insecure-unlock 

2 окно подключение 2 окна к первому окну:   

geth attach "//./pipe/geth.ipc"

2 окно создаем аккаунт:
  
personal.newAccount()   при создании у вас появиться аккаунт по типу: 0xe19f085234596ac12566cf3a061a94f3a39ccac5

Дополнительная информация во 2 окне:
miner.start() - функция запустить майнинг
miner.stop() -   функция остановить майнер
eth.accounts - узнать аккаунты
eth.getBalance(eth.accounts[0]) - баланс кошелька
eth.accounts[0] - узнать номер аккаунта
miner.setEtherbase(eth.accounts[0]) - смена кошелька для майнинга


personal.unlockAccount(eth.accounts[0], "пароль") - разблокировка аккаунта для выполнения операций
eth.sendTransaction({from: eth.accounts[0], to: eth.accounts[1], value: web3.toWei(2 , 'ether')}) - первод эфира
eth.sendTransaction({from: "номер кошелька", to: "номер кошелька", value: web3.toWei(1, 'ether')})
